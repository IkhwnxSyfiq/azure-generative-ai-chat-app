# Azure Generative AI Chat App

## Overview

This project is based on a hands-on lab completed while preparing for the **Azure AI Engineer professional certification** using Microsoft Learn.

The lab demonstrates how to build a **chat application that leverages Azure OpenAI models** to provide intelligent, context-aware responses in a conversational interface.

## What I Learned

* Deploying and managing Azure OpenAI models
* Implementing chat completion with appropriate system prompts and temperature settings
* Handling conversation history for context-aware responses
* Building a simple chat interface using Python
* Managing tokens and optimising model performance

## Technologies Used
* Azure OpenAI (GPT-4o models)
* Python
* Azure AI Foundry

## Purpose
The purpose of this project is to document my hands-on experience with Azure OpenAI services while learning how to develop Generative AI chat applications that can engage in natural, helpful conversations.

## Note
This project was completed as part of a hands-on lab during my preparation for the **Azure AI Engineer certification**.
